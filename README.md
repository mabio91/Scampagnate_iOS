# Scampagnate iOS

App iOS nativa di Scampagnate.

## Repository

Questa repo contiene solo il progetto iOS. La web app user, l'admin e l'eventuale app Android restano in repository separati per permettere sviluppi paralleli senza mischiare ambienti e dipendenze.

## Note operative

- Non committare certificati, provisioning profile, chiavi private o file `.env`.
- Le modifiche che impattano Supabase/API devono essere coordinate anche con le repo web e admin.
- Il branch principale e' `main`.
